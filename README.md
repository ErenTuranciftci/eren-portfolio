# Eren Turançiftçi - Portföy Sitesi

Daha doğru test için VS Code'da **Live Server** eklentisini kullanın:
1. Klasörü VS Code ile açın.
2. `index.html` dosyasına sağ tıklayın.
3. **Open with Live Server** seçeneğini seçin.

## Düzenleme

- Metinler ve bağlantılar: `index.html`
- Renkler, boşluklar ve responsive tasarım: `styles.css`
- Mobil menü ve kaydırma animasyonu: `script.js`
- İndirilebilir CV: `assets/Eren-Turanciftci-CV.pdf`
