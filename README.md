# Eren Turançiftçi - Portföy Sitesi

Bu proje yalnızca HTML, CSS ve JavaScript kullanır. Backend veya veritabanı gerektirmez.

## Bilgisayarda çalıştırma

En kolay yöntem, klasörde `index.html` dosyasına çift tıklamaktır.

Daha doğru test için VS Code'da **Live Server** eklentisini kullanın:
1. Klasörü VS Code ile açın.
2. `index.html` dosyasına sağ tıklayın.
3. **Open with Live Server** seçeneğini seçin.

## Düzenleme

- Metinler ve bağlantılar: `index.html`
- Renkler, boşluklar ve responsive tasarım: `styles.css`
- Mobil menü ve kaydırma animasyonu: `script.js`
- İndirilebilir CV: `assets/Eren-Turanciftci-CV.pdf`

## GitHub Pages ile yayınlama

1. GitHub'da yeni ve public bir repository oluşturun.
2. Bu klasörün içindeki dosyaları repository'nin ana dizinine yükleyin.
3. Repository'de **Settings > Pages** bölümünü açın.
4. Source alanında **Deploy from a branch** seçin.
5. Branch olarak `main`, klasör olarak `/ (root)` seçip kaydedin.
6. Birkaç dakika sonra site `https://kullaniciadi.github.io/repository-adi/` adresinde açılır.

## Netlify ile yayınlama

1. Netlify hesabınıza giriş yapın.
2. **Add new site > Deploy manually** seçeneğini açın.
3. `eren-portfolio` klasörünü ekrana sürükleyip bırakın.
4. Netlify otomatik bir adres oluşturur.

## Kişisel alan adı

Bir alan adı satın aldıktan sonra GitHub Pages veya Netlify panelindeki **Custom Domain** bölümünden bağlayabilirsiniz.
